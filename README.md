# Craftbuild
A Minecraft likes Voxel game made with C++23 and Godot.

# Status
Indev

# Features
- Enviroment
- Generate terrain
- Save/Load world
- Collision, Camera, Skin
- Break block
- Menu
- Pause screen

# Comming soon
- Hotbar
- Place block
- Lighting system
- Cloud
- Weather
- Multiplayer and networking
