# Craftbuild
A Minecraft-like Voxel game made with C++23 and Godot.

# Status
Indev

# Features
- Environment
- Generate terrain
- Save/Load world
- Collision, Camera, Skin
- Break block
- Menu
- Pause screen
- Multi-threading
- Greedy meshing, texture LOD

# Coming soon
- Hotbar
- Place block
- Lighting system
- Cloud
- Weather
- Multiplayer and networking
